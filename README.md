Yo, guys. This`s my Arch Linux + i3-gaps ricing setup. Soon I will make the installation file/docs for people. This repository is always being updated, so stay tuned!

- Window Manager: [i3-gaps](https://i3wm.org/)
- IDE / Text Editor: [LunarVim](https://www.lunarvim.org/)
- Bar:  [Polybar](https://polybar.github.io/)
- Shell: [zsh](https://ohmyz.sh/)
- Compositor: [Picom](https://github.com/yshui/picom)
- Terminal & File Manager: [alacritty](https://alacritty.org/)

- Polybar Themes: [themes](https://github.com/adi1090x/polybar-themes)
